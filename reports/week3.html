Week 03 — Simulation

Microcontroller Programming & Simulations

This week we transitioned from general programming to hardware-specific logic. Before deploying code to real silicon, we utilized industry-standard simulation tools to validate our logic and wiring diagrams.

Learning Objectives

  • Microcontroller Architecture: Memory mapping and I/O registers.
  • Practical Simulation using Wokwi and Tinkercad.
  • Developing Pushbutton and LED logic.
  • Debugging embedded systems in virtual environments.

Simulation Workflow

Working with physical hardware can be risky—short circuits or incorrect voltages can permanently damage components. Tools like Wokwi and Tinkercad allow us to build circuits virtually. We simulated the Arduino Uno R3 and Raspberry Pi Pico, focusing on how external signals interact with internal firmware.

Assignment: Pushbutton & LED Control

The goal was to create a circuit where an LED is toggled by a physical pushbutton, incorporating debouncing logic to ensure stable state transitions.
