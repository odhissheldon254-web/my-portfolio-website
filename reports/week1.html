Week 01 — Introduction

Introduction to Embedded Systems

My journey into the world of embedded systems began with a deep dive into the synchronization of hardware and software. This week focused on setting up the fundamental tools and understanding the core architecture that drives modern instrumentation.

Learning Objectives

  • Foundations of Embedded System Architecture.
  • Introduction to Web Technologies (HTML/CSS) for UI development.
  • Python for Rapid Prototyping in Microcontrollers.
  • Development Environment Setup (Thonny & Arduino IDE).
  • Hardware Overview: Raspberry Pi Pico (RP2040).

Key Insights

We explored the RP2040 chip, which features a dual-core ARM Cortex-M0+ processor. Understanding its GPIO layout and the versatility of MicroPython allows for incredibly fast iteration cycles compared to traditional low-level C for simple tasks.

I also began structuring this portfolio using semantic HTML5 and custom CSS variables, ensuring a modular and highly interactive user experience from the start.

Assignment: Python Area Calculator

The goal was to create a flexible script that calculates the area of different geometric shapes. I implemented this using a modular approach with input validation to ensure robustness.

# Geometric Area Calculator import math def calculate_area(): while True: print("\n--- Area Calculator ---") print("1. Circle") print("2. Square") print("3. Rectangle") print("4. Exit") choice = input("Select shape (1-4): ") if choice == '1': radius = float(input("Enter radius: ")) area = math.pi * (radius ** 2) print(f"Area: {area:.2f}") elif choice == '2': side = float(input("Enter side length: ")) print(f"Area: {side * side:.2f}") elif choice == '3': l = float(input("Enter length: ")) w = float(input("Enter width: ")) print(f"Area: {l * w:.2f}") elif choice == '4': break else: print("Invalid selection.") if __name__ == "__main__": calculate_area()
